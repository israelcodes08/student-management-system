# student-management-system

## Description
This is a simple Python applicaion that allows users to manage student records. Users can add students, and store basic information such as names and grades.

## Features 
- Add student names and grades
- View all stored students
- Simple menu-based interface

## Technologies Used 
- Python

## How to Run
1. Open the file in a Python environment (e.g. Thonny or Visual Studio Code)
2. Run the program
3. Follow the on-screen menu
